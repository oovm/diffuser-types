如果一行的首个非空符号为 `#`, 那么这行会作为一个注释

如果一行结尾没加 `,`, 那么会自动加上 `,`

`( )` 表示强调, `[ ]` 表示减弱, `\( \)` 表示转义, 不强调

| Input          | Word   | Weight |
|----------------|--------|--------|
| `(word)`       | word   | 1.1    |
| `((word))`     | word   | 1.21   |
| `[word]`       | word   | 0.9    |
| `(word:1.5)`   | word   | 1.5    |
| `(word:0.25)`  | word   | 0.25   |
| `\(word\)`     | (word) | 1.1    |
| `\(word\)`     | (word) | 1.1    |
| `((word:1.5))` | word   | 1.95   |
